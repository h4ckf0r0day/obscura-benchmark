# Real-world scrape-target corpus for the obscura render-success benchmark.

# These are pages people actually scrape, across the categories real scraping
# workloads hit. The live web drifts, so success rates are a snapshot.
#
# IMPORTANT: run this with `--chrome` (obscura vs headless Chrome side by side).
# Many of these sites block datacenter IPs (anti-bot / CAPTCHA / login walls), so
# from a server they fail for EVERY engine. A failure only indicts obscura's
# engine when Chrome succeeds on the same URL and obscura does not. Same-IP
# Chrome comparison separates "engine gap" from "site blocks everyone".
#
# One URL per line; blank lines and lines starting with # are ignored.
